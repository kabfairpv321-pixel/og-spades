# OG Spades

**The spades app the big ones never built.**

Partners spades, built honest. Real stats. 30 achievements. 6 card decks. Multiple table themes. No coin grinds.

## What it is

OG Spades is a single-player partners spades game (you + bot partner vs two bot opponents) with persistent stats, achievements, customizable rule variants, custom avatars, and a deck/theme picker.

## Features

- Full partners spades — bidding, trick-taking, scoring, bag penalties
- 9 customizable rule variants (score target 100-500, high 2♠, jokers, blind nil, bot difficulty)
- 30 achievements across 4 tiers
- 6 card decks (4 free, 2 premium-locked)
- 2 table themes (Classic Felt, Holodeck)
- Custom avatar builder (9 parts) or photo upload
- Stats tracking (wins, streaks, nils, bostons, comebacks)
- Sound system (toggleable)
- Persistent profile and progress

## Development

```bash
npm install
npm run dev
```

## Tech

- React 18
- Vite
- Tailwind CSS
- Web Audio API (synthesized sounds, no audio files)
- localStorage for persistence
